# GoStore
Basic E-Commerce Web App built with GO

# Features
- Home Page
- Register, Login, Profil User
- Products Page
- Single Product Page
- Shopping Cart
- CheckOut
  - Shiping Cost Calculation
  - Payment Gateway
- Order History
- Admin Panel :
  - Dashboard
  - Manage Products
  - Manage Users
  - Reports
  - Configuration

# Technology Stack
- Golang :
  - Gorilla Mux
  - Gorm
- PostgreSQL
- Bootstrap
